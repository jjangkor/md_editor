# JJANG's MD Editor

JJANG's MD Editor - 별도의 빌드 과정 없이 즉시 실행 가능한 단일 파일 마크다운 에디터입니다.

![Markdown Editor](https://img.shields.io/badge/Markdown-Editor-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Build](https://img.shields.io/badge/build-standalone-brightgreen)

## 주요 특징

- ✅ **단일 HTML 파일**: 모든 코드가 하나의 `md_editor.html` 파일에 포함
- ✅ **빌드 불필요**: 브라우저에서 바로 실행 가능
- ✅ **CDN 기반**: 모든 라이브러리를 CDN으로 로드하여 별도 설치 불필요
- ✅ **반응형 레이아웃**: 고정 100vh 레이아웃으로 스크롤 없는 깔끔한 UI
- ✅ **실시간 미리보기**: 입력과 동시에 마크다운 렌더링
- ✅ **스크롤 동기화**: 에디터와 프리뷰 창의 스크롤 자동 동기화
- ✅ **자동 목차 생성**: `&lt;!-- TOC --&gt;` 주석으로 자동 목차 생성
- ✅ **수학 수식 지원**: LaTeX 문법으로 수식 작성 및 KaTeX 렌더링
- ✅ **크기 조절 가능**: 에디터와 프리뷰 창 크기를 자유롭게 조절
- ✅ **계층적 리스트**: 불릿(□→○→●)과 자동 번호(1→1.1→1.1.1) 지원
- ✅ **찾기/바꾸기**: 텍스트 검색 및 일괄 치환 기능 (Ctrl+F, Ctrl+H)
- ✅ **다양한 내보내기**: PDF, Word 형식 지원 (저장 경로 선택 대화창)
- ✅ **파일 관리**: 마크다운 파일 열기/저장 기능

## 기능 목록

### 1. 실시간 마크다운 미리보기
- GitHub Flavored Markdown (GFM) 완벽 지원
- 입력 즉시 오른쪽 프리뷰 영역에 렌더링
- Syntax Highlighting 지원

### 2. 스크롤 동기화
- 왼쪽 에디터를 스크롤하면 오른쪽 프리뷰도 자동으로 스크롤
- 오른쪽 프리뷰를 스크롤하면 왼쪽 에디터도 자동으로 스크롤
- 스크롤 비율을 기준으로 정확하게 동기화
- 무한 루프 방지 메커니즘으로 부드러운 스크롤 제공

### 3. 파일 관리

#### 파일 열기
- **버튼 클릭**: 상단 "파일 열기" 버튼으로 .md, .markdown, .txt 파일 선택
- **드래그 앤 드롭**: 에디터 영역에 파일을 드래그하여 바로 열기
- 파일 열기 시 자동으로 파일명 표시 및 히스토리 초기화

#### 파일 저장 및 내보내기
- **MD 저장**: 현재 내용을 마크다운(.md) 파일로 저장 (저장 경로 선택 대화창)
- **PDF**: 프리뷰를 PDF 문서로 내보내기 (저장 경로 선택 대화창)
  - 페이지 나누기 최적화: 단락, 제목, 리스트 항목이 페이지 중간에 잘리지 않음
- **Word**: .doc 형식으로 내보내기 (저장 경로 선택 대화창)
- File System Access API 미지원 브라우저에서는 자동 다운로드로 폴백

### 4. 자동 목차 생성

문서에 `&lt;!-- TOC --&gt;` 주석을 추가하면 자동으로 목차가 생성됩니다.

```markdown
# 문서 제목

<!-- TOC -->
자동으로 생성될 위치
<!-- /TOC -->

## 섹션 1
### 하위 섹션 1.1

## 섹션 2
```

- 모든 헤딩(#, ##, ###...)을 자동으로 감지
- 들여쓰기가 적용된 계층적 목차 생성
- 클릭 가능한 앵커 링크로 변환
- 실시간 업데이트 (헤딩 추가/삭제 시 자동 반영)

### 5. 수학 수식 지원

LaTeX 문법으로 수학 수식을 작성하면 KaTeX로 자동 렌더링됩니다.

#### 인라인 수식
`$...$` 구문을 사용하여 텍스트 내에 수식 삽입:
```markdown
질량-에너지 등가 원리는 $E = mc^2$ 입니다.
```

#### 블록 수식
`$ ... $` 구문을 사용하여 수식 작성 (줄바꿈 지원):
```markdown
$ \int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi} $

$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $
```

#### 지원 기능
- **복잡한 수식**: 분수, 적분, 합, 행렬 등 모든 LaTeX 수식 지원
- **줄바꿈 지원**: `$ ... $` 사이에 줄바꿈 가능 (복잡한 수식용)
- **그리스 문자**: α, β, γ, Δ, Σ 등
- **특수 기호**: ∞, ±, ≤, ≥, ∈, ∀, ∃ 등
- **실시간 렌더링**: 입력 즉시 수식 프리뷰
- **에러 처리**: 잘못된 수식도 안전하게 표시

#### 수식 예제
```markdown
이차방정식의 해:
$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $

피타고라스 정리: $a^2 + b^2 = c^2$

행렬 (여러 줄):
$
\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}
$
```

### 6. 강력한 편집 기능

#### 툴바 버튼
- **되돌리기/다시실행**: Undo/Redo 기능으로 편집 히스토리 관리
- **텍스트 서식**: 굵게, 기울임
- **제목**: H1, H2 헤더
- **목록**: 순서 있는 목록, 순서 없는 목록
- **코드**: 인라인 코드, 코드 블록
- **인용구**: 블록 인용
- **목차**: 자동 목차 주석 삽입
- **수식**: 수학 수식 삽입 ($E = mc^2$)

#### 키보드 단축키
- `Ctrl+Z` / `Cmd+Z`: 되돌리기
- `Ctrl+Y` / `Cmd+Y` 또는 `Ctrl+Shift+Z`: 다시실행
- `Ctrl+B` / `Cmd+B`: 굵게
- `Ctrl+I` / `Cmd+I`: 기울임
- `Ctrl+F`: 찾기
- `Ctrl+H`: 찾아 바꾸기

### 7. 찾기 및 바꾸기

#### 찾기 기능
- **텍스트 검색**: 에디터에서 단어나 문구 검색
- **실시간 검색**: 입력하는 즉시 검색 결과 표시
- **매치 카운트**: 찾은 항목 개수 표시 (예: "3/10")
- **탐색 기능**: 이전/다음 버튼으로 검색 결과 이동
- **자동 하이라이트**: 현재 위치 자동 선택 및 스크롤

#### 바꾸기 기능
- **단일 바꾸기**: 현재 선택된 항목 하나만 바꾸기
- **전체 바꾸기**: 모든 매치 항목을 한 번에 바꾸기
- **히스토리 지원**: 바꾸기 후 되돌리기(Undo) 가능

#### 키보드 단축키
- `Ctrl+F`: 찾기 패널 열기
- `Ctrl+H`: 찾아 바꾸기 패널 열기
- `Enter`: 다음 항목 찾기
- `Shift+Enter`: 이전 항목 찾기
- `Esc`: 패널 닫기

### 8. 크기 조절 가능한 분할 창
- 에디터와 프리뷰 사이의 구분선을 드래그하여 비율 조정
- 최소 20%, 최대 80%까지 조절 가능
- 마우스 커서 변경으로 직관적인 UX 제공

### 9. 히스토리 관리
- 최대 100개의 편집 히스토리 저장
- 디바운싱(500ms)으로 효율적인 히스토리 관리
- 커서 위치까지 정확하게 복원

## 사용 방법

### 기본 사용

1. **마크다운 파일 열기**
   - 상단의 "파일 열기" 버튼 클릭하여 .md 파일 선택
   - 또는 에디터 영역에 파일을 드래그 앤 드롭
   - 파일명이 헤더에 자동으로 표시됨

2. **마크다운 작성**
   - 왼쪽 에디터 창에 마크다운 텍스트 입력
   - 오른쪽 프리뷰 창에서 실시간으로 결과 확인

3. **목차 추가하기**
   - 툴바에서 목차 버튼(리스트 아이콘) 클릭
   - 또는 직접 `&lt;!-- TOC --&gt;` 와 `&lt;!-- /TOC --&gt;` 주석 작성
   - 자동으로 문서의 모든 헤딩이 목차로 생성됨

4. **서식 적용**
   - 텍스트를 드래그하여 선택
   - 툴바 버튼 클릭으로 마크다운 문법 자동 삽입
   - 또는 키보드 단축키 사용

5. **창 크기 조절**
   - 중앙의 회색 구분선을 마우스로 드래그
   - 원하는 비율로 에디터/프리뷰 크기 조정

6. **문서 저장**
   - **MD 저장**: 마크다운 파일로 저장
   - **PDF**: PDF 문서로 내보내기
   - **Word**: Word 문서(.doc)로 내보내기

### 마크다운 문법 예제

```markdown
# 제목 1
## 제목 2

**굵은 텍스트**
*기울임 텍스트*
~~취소선 텍스트~~

- 순서 없는 목록 (□)
- 항목 2
  - 중첩 항목 (○)
    - 3단계 항목 (●)

1. 순서 있는 목록 (1로만 써도 자동 번호 매김)
1. 항목 2
   1. 중첩 항목 (1.1로 표시)
   1. 항목 2-2 (1.2로 표시)
      1. 3단계 항목 (1.2.1로 표시)
1. 항목 3

`인라인 코드`

\`\`\`javascript
// 코드 블록
function hello() {
  console.log("Hello, World!");
}
\`\`\`

> 인용구
```

#### 리스트 스타일
- **불릿 리스트**: 1단계 □ (사각형) → 2단계 ○ (원형) → 3단계 ● (채움)
- **순서 리스트**: 1단계 1, 2, 3... → 2단계 1.1, 1.2... → 3단계 1.1.1, 1.1.2...
- 모든 항목을 `1.`로 작성해도 자동으로 번호가 매겨집니다

## 코드 아키텍처

### 주요 컴포넌트

1. **수학 수식 렌더링 시스템**
   - `renderMath()`: KaTeX를 사용한 LaTeX 수식 렌더링
   - 인라인 수식 (`$...$`) 및 블록 수식 (`$$...$$`) 지원
   - 실시간 수식 프리뷰
   - 에러 처리 및 안전한 fallback

2. **자동 목차 생성 시스템**
   - `generateTOC()`: 마크다운 헤딩을 파싱하여 목차 생성
   - `&lt;!-- TOC --&gt;` 주석 감지 및 자동 교체
   - 계층적 들여쓰기와 앵커 링크 생성
   - 실시간 업데이트 및 한글/영문 지원

3. **파일 관리 시스템**
   - `openMarkdownFile()`: FileReader API를 사용한 파일 읽기
   - `saveMarkdownFile()`: Blob API를 사용한 파일 저장
   - `setupDragAndDrop()`: 드래그 앤 드롭 이벤트 처리

4. **히스토리 관리 시스템**
   - `saveToHistory()`: 현재 상태를 히스토리에 저장
   - `undo()` / `redo()`: 되돌리기/다시실행
   - `restoreState()`: 이전 상태 복원
   - 최대 100개 히스토리 유지, FIFO 방식

5. **마크다운 에디터**
   - `insertMarkdown()`: 커서 위치에 마크다운 문법 삽입
   - 텍스트 선택 시 자동으로 감싸기
   - 커서 위치 자동 조정
   - TOC 및 수식 주석 자동 삽입

6. **리사이저**
   - 마우스 드래그 이벤트 처리
   - Flexbox 기반 동적 크기 조절
   - 20%~80% 범위 제한

7. **내보내기 기능**
   - `exportToPDF()`: html2pdf.js를 사용한 PDF 생성
   - `exportToWord()`: Blob API를 사용한 .doc 생성

### 에러 처리

- CDN 로드 실패 감지 및 사용자 알림
- 마크다운 파싱 오류 처리
- 수식 렌더링 오류 안전 처리
- PDF/Word 생성 실패 시 알림
- 파일 읽기/쓰기 오류 처리

## 특별 기능: 마크다운 파일 연결

### Windows에서 설정

1. .md 파일 우클릭 → "연결 프로그램" → "다른 앱 선택"
2. "이 PC에서 다른 앱 찾기" 클릭
3. `md_editor.html` 파일 선택
4. "항상 이 앱을 사용하여 .md 파일 열기" 체크

### macOS에서 설정

1. .md 파일 우클릭 → "정보 가져오기"
2. "다음으로 열기" 섹션에서 브라우저 선택
3. "모두 변경..." 클릭

- 브라우저 보안 정책상 로컬 파일 시스템 직접 접근은 제한됨
- 파일 선택 또는 드래그 앤 드롭 방식으로 파일 열기 권장
